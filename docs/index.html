üìñ API Reference

lookupMedication(query: string): string

Returns formatted markdown with medication info, dosage, warnings, and FASS link. Perfect for chatbot responses.

findMedication(query: string): object | null

Returns raw medication data object for programmatic use, or null if not found in the quick-lookup database.

getFassUrl(query: string): string

Returns the FASS.se search URL for any medication query.

COMMON_MEDICATIONS: object

The raw medications database. Contains ~10 common Swedish medications with full details.
